第三章：手把手导入教学（图文详解）

3.1 基础导入流程

步骤1：准备阶段

  • 确保Quantumult为最新版本（App Store检查更新）
  • 准备好节点链接或配置文件

步骤2：链接导入法

  1. 打开Quantumult → 点击右下角"配置"
  2. 选择"节点" → 点击"+"号
  3. 选择"从URL导入" → 粘贴节点链接
  4. 点击"完成" → 返回主界面测试连接

步骤3：文件导入法（适合批量操作）

  1. 将节点文件保存至iCloud Drive
  2. 在Quantumult中选择"从文件导入"
  3. 定位并选择节点文件
  4. 解析完成后检查节点列表

3.2 高级技巧：订阅自动更新

配置方法：

  1. 获取订阅链接（通常以"https://"开头）
  2. 进入"配置" → "订阅" → 添加订阅
  3. 设置更新频率（建议6-12小时）

优势分析：

  • 自动获取最新节点，免去手动更新麻烦
  • 节点失效时自动切换，保障持续连接

第四章：节点优化与管理策略

4.1 节点性能测试

测试方法：

  1. 长按节点 → 选择"延迟测试"
  2. 解读结果：
    • <100ms：极佳（适合视频通话、游戏）
    • 100-300ms：良好（常规浏览、视频流畅）
    • >300ms：考虑更换

4.2 智能分流配置

推荐规则：

  • 国内直连：微信、支付宝等常用APP
  • 国外代理：Google、YouTube等国际服务
  • 广告拦截：内置规则更新

第五章：安全防护与故障排除

5.1 安全使用守则

  • 警惕"永久免费"节点（可能收集用户数据）
  • 重要账户避免通过陌生节点登录
  • 定期检查连接日志（路径：配置→日志）

5.2 常见问题解决方案

问题1：导入后无法连接

  • 检查项：
    ✓ 节点是否已过期
    ✓ 本地网络是否正常
    ✓ 协议类型是否匹配

问题2：速度突然下降

  • 应对措施：
    1. 切换不同地区节点
    2. 检查是否为网络高峰期
    3. 联系服务商确认带宽状态
